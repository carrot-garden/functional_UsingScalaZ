Actually Using ScalaZ

Slides and code for a presentation about using ScalaZ.

View on GitHub Download .zip Download .tar.gz

This project contains the presentation I gave to the Chicago Area Scala Enthusiasts (CASE) meeting on May 21, 2015. The topic of the talk was "Actually using ScalaZ". The target audience is developers who have been around Scala long enough to have encountered ScalaZ and been scared away by it.

The slides are available here. Pressing the 's' key while viewing the presentation will open the speaker view and make my speaker notes visible. I've tried to put enough detail into the notes to cover what I intended to say about each slide.
